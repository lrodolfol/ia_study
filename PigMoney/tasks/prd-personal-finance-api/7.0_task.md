## Overview

Implement the API layer infrastructure including middleware for response standardization and global exception handling, configure dependency injection for all services and repositories, setup the middleware pipeline, configure CORS, and implement structured logging. This task establishes the foundation for all API endpoints.

- Setup Microsoft.Extensions.Logging with structured logging

Key implementation points:
- ResponseWrapperMiddleware:
  ```csharp
  public async Task InvokeAsync(HttpContext context)
  {
      var originalBodyStream = context.Response.Body;
      using var responseBody = new MemoryStream();
      context.Response.Body = responseBody;
      
      await _next(context);
      
      // Wrap response in StandardApiResponse format
      var response = new StandardApiResponse<object>(data, errors, message, statusCode);
      await WriteResponseAsync(context, response);
  }
  ```
- ExceptionHandlerMiddleware: Catch all exceptions, log with correlation ID, return 500 with sanitized message
- DI registration:
  ```csharp
  builder.Services.AddDbContext<AppDbContext>(options =>
  ```
